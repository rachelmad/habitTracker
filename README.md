# HabitTracker
A MERN (Mongo, Express, React, Node) site to list, track, and help users maintain habits. 

## To clone:

```
git clone 
cd habitTracker
npm install
```

## Commands to keep running:
```
node webapp.js
```
```
./node_modules/babel-cli/bin/babel.js --presets react --watch src --out-dir static
```
